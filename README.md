# VibeVision

VibeVision is a simple movie‑themed web app built with React and Firebase. It includes user login, protected pages, and a clean UI for browsing movie content.

---

## Features

- **Authentication:** Google login with Firebase  
- **Security:** Protected routes for logged‑in users  
- **Navigation:** React Router for page routing  
- **Data:** Firestore database support  
- **UI:** Responsive layout and reusable components  

---

## Tech Used

- **Frontend:** React, React Router, JavaScript, HTML, CSS  
- **Backend / Services:** Firebase Auth, Firestore  
- **Tooling:** Vite, npm
